# ✅ Gerenciador de Tarefas

Um **Gerenciador de Tarefas** simples e intuitivo para criar, visualizar e remover tarefas, com campos de **título** e **descrição**.

O projeto foi desenvolvido utilizando **React** e **TailwindCSS**, com interface moderna e responsiva.

🔗 **Acesse o projeto online:**  
https://gerenciador-tarefas-git-main-wagner53426s-projects.vercel.app/

---

# ✨ Funcionalidades

- ➕ Adicionar uma nova tarefa
- 📝 Adicionar descrição para cada tarefa
- 📋 Visualizar lista de tarefas
- 🗑️ Remover tarefas
- 🎨 Interface moderna e simples
- ⚡ Navegação utilizando React Router
